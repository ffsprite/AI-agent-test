# Multi Agent Collaboration System

A complete multi-agent AI workflow system built with Python and OpenAI API.

## Features

- Planner Agent
- Research Agent
- Coding Agent
- Review Agent
- Manager Workflow
- LangGraph Support

## Installation

```bash
pip install -r requirements.txt
```

## Setup

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key
```

## Run

```bash
python main.py
```
